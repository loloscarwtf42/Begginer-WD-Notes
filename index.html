Images

IMG adds images, theese have to include the atribute src="", this stands for SOURCE. The source can be directed to a specific location on the PC (desktop/folder/image.jpg), or to a link or address like the following image. The atribute alt="" stands for an alternative description. You'll be able to see this description when you hover over the image.

The atributes width and height are used to resize the image. Theese can be represented in Pixels (px) or Percentages (%). This image has a 30% width atribute, this means that the height will be automaticly scaled and the width of the image will always be 30% of the window.

Links

A, is for links. The atribute href="" gives it a directory and the the text between the A tag can be whatever you want. The directory of the atribute can be either a link to a diferent page, or a location on your own PC (desktop/folder/index.html). The A tag can be inserted in any text, such as paragraphs, lists...

Tables

Table, is used to make tables. Inside the table tag, every TR is a table row, and each will be one on top of the other.
Every TD is a table data or cell tag, and they will be positioned from left to right inside every TR.

Tables have also got thead, tbody, tfoot tags on the inside. The Table Header uses inner tags such as th.
This separations are used to give tables diferent organizations and styles that'll be used on css mainly.

We can add atributes such as border="" filled with px, or cellspacing="" for other views of it.

Forms

Form is used to make up forms for users to complete and submit.
Inside the Form tag, there are diferent tags and atributes to be used.
Label is situated between this tag and it's used to give a name or lable to te diferent tasks of the form.
Input is located on the same place and it's used to create a space to complete an action (type, submit...). This tag uses the atribute type="", and can be filled with diferent uses (text, sumbit).

Theese are different variations of the input tag. More uses can be found on the HTML MDN page.

Textarea is used to make a large space to type in within your form. This can be modified changing the columns or the rows through the atributes on the tag.

In order to use this form a to send an email, we'll have to change the atributes within the form tag.
Theese are the atrifutes in the form tag: class="" action="index.html" method="post"
It shold look like this: action="mailto:whoeveruwant@gmail.com" method="post" enctype="text/plain", plus, the name atribute in every input should have its own description.
